O que é HTML?

O HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar páginas na web. Ele organiza e estrutura o conteúdo que aparece nos navegadores, como textos, imagens, links e mais, utilizando elementos chamados TAGs.

(Voltar)

O que são TAGs?

TAGs são comandos especiais usados para organizar o conteúdo da página. Elas possuem abertura (<tag>) e fechamento (</tag>), com algumas exceções como <hr>.

(Voltar)

<html> ... </html>

Envolve todo o conteúdo da página. É o elemento raiz, ou seja, todos os demais elementos da página devem estar inseridos entre a abertura e fechamento dessa tag.

(Voltar)

<head> ... </head>

Contém informações sobre a página, como título e metadados.

(Voltar)

<title> ... </title>

Define o título que aparece na aba do navegador.

Exemplo: <title>Título da Minha Página</title>

(Voltar)

<body> ... </body>

Contém o conteúdo visível da página, como textos e imagens.

Exemplo: <body>conteúdo (textos, imagens, etc...)</body>

(Voltar)

<h1> ... </h1> (Obs.: até h6)

Representam títulos em ordem de importância. Podem ser de "h1" até "h6"

Exemplos:

  • <h1> Título Principal </h1>
  • <h2> Subtítulo </h2>
(Voltar)

<p> ... </p>

Define parágrafos.

Exemplo: <p>Digite o conteúdo do parágrafo ...</p>

(Voltar)

<mark> ... </mark>

Destaca texto, simulando um "marca-texto"

Exemplo: <mark>Texto a ser destacado</mark> (Voltar)

<small> ... </small>

Exibe texto menor em relação ao texto.

Exemplo: <small>Texto em fonte reduzida</small>

(Voltar)

<i> ... </i>

Aplica itálico ao texto

Exemplo: <i>Texto em itálico</i>

(Voltar)

<u> ... </u>

Sublinha o texto

Exemplo: <u>Texto sublinhado</u>

(Voltar)

<strong> ... </strong>

Aplica negrito com destaque/ênfase

Exemplo: <strong>Digite o texto ...</strong>

(Voltar)

<ol> ... </ol>

Cria uma lista ordenada numericamente. Para inserir itens na lista, utliza-se a tag <li> ... </li>

Exemplo:

  1. item da lista
  2. item da lista
  3. item da lista
(Voltar)

<ul> ... </ul>

Cria uma lista não ordenada. Para inserir itens na lista, utliza-se a tag <li> ... </li>

Exemplo:

  • item da lista
  • item da lista
  • item da lista

(Voltar)

<li> ... </li>

Representa itens de uma lista ordenada ou não ordenada

(Voltar)

<a> ... </a>

Cria links para outras páginas

O atributo href indica a referêcia destinatária

Exemplo: <a href="https://g1.globo.com/">Notícas do Site G1</a> (Voltar)

<hr>

Insere uma linha horizontal.

(Voltar)

<sub> ... </sub>

Texto em subscrito

Exemplo: H<sub>2</sub>O = H2O.

(Voltar)

<sup> ... </sup>

Texto em sobrescrito

Exemplo: 10<sup>2</sub> = 102.

(Voltar)

<blockquote> ... </blockquote>

Indica uma citação.

Exemplo:

Não há caminho para a paz; a paz é o caminho. (Mahatma Gandhi)

Codificação:

<blockquote>Não há caminho para a paz; a paz é o caminho. (Mahatma Gandhi)</blockquote>

(Voltar)

<font> ... </font>

Personaliza fontes. Usada para alterar características de texto como a cor, o tamanho e o tipo de fonte. Contudo, ela está obsoleta e não é recomendada em projetos modernos, pois o uso de CSS é mais eficiente para personalização. Utiliza atributo como "color", "size" e "face"

Exemplo: Texto em cor azul, tamanho 4, na fonte Arial

Codificação:

Texto em <font color="blue" size="4", face="Arial">cor azul, tamanho 4, na fonte Arial</font>

(Voltar)

<del> ... </del>

Exibe texto riscado.

Exemplo: <del>digite o texto a ser riscado</del>

(Voltar)

<abbr> ... </abbr>

Define abreviações com significados. Ao passar o mouse sobre o item, aparecerá um Um tooltip é uma pequena caixa de texto que aparece quando o usuário passa o cursor do mouse sobre um elemento na tela. com a descrição.

Exemplo: Os EUAsão uma das maiores potências econômicas e militares no mundo contemporâneo.
